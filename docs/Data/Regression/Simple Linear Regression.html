Simple Linear Regression

Relationship between Two Numerical Variables​

When we have two numerical variables, say XXX and YYY, we are typically interested in whether they are correlated - that is, how YYY changes as XXX changes.

  • Positive association: As the value of XXX increases, YYY tends to increase as well (or vice versa).
  • Negative association: As the value of XXX decreases, YYY tends to increase (or vice versa).

Data​

(x1,Y1),⋯ ,(xn,Yn)(x_{1},Y_{1}), \cdots, (x_{n},Y_{n})(x1​,Y1​),⋯,(xn​,Yn​)

  • YiY_{i}Yi​ : Response/Dependent variable. Random variable.
  • xix_{i}xi​ : Independent/Expalanatory/Predictor/Regressor variable. Fixed quantity.

Simple Linear Regression Model​

Yi=β0+β1xi+ϵi,Y_{i} = \beta_{0} + \beta_{1}x_{i}+\epsilon_{i},Yi​=β0​+β1​xi​+ϵi​,

where ϵi∼iid(0,σ2)\epsilon_{i} \overset{iid}{\sim} (0,\sigma^{2})ϵi​∼iid(0,σ2)

  • E(Yi)=E(β0+β1xi+ϵi)=β0+β1xiE(Y_{i}) = E(\beta_{0} + \beta_{1}x_{i}+\epsilon_{i}) = \beta_{0} + \beta_{1}x_{i}E(Yi​)=E(β0​+β1​xi​+ϵi​)=β0​+β1​xi​
  • Var(Yi)=Var(β0+β1xi+ϵi)=Var(ϵi)=σ2Var(Y_{i}) = Var(\beta_{0} + \beta_{1}x_{i}+\epsilon_{i}) = Var(\epsilon_{i}) = \sigma^{2}Var(Yi​)=Var(β0​+β1​xi​+ϵi​)=Var(ϵi​)=σ2

Parameters​

  • β0\beta_{0}β0​ : Unknown Intercept, the expected value of YYY when X=0X=0X=0.
  • β1\beta_{1}β1​ : Unknown Slope, represents the change of YYY when XXX increase by 1 Unit.
    • Posses an important information for the relationship between XXX and YYY.
  • σ2\sigma^{2}σ2 : Error variance plays an important role for statistical inference
    • e.g. Hypothesis testing, confidence intervals, …\ldots…

Key Assumptions​

  1. Linearity
    • XXX and YYY have a linear relationship.
  2. Independence
    • The error terms, ϵi\epsilon_{i}ϵi​, are independent.
  3. Constant variance
    • Var(ϵi)=σ2⇒ϵiVar(\epsilon_i) = \sigma^2 \Rightarrow \epsilon_{i}Var(ϵi​)=σ2⇒ϵi​ are identically distributed.
  4. Normality (additional assumption)
    • ϵi\epsilon_{i}ϵi​ are Normally distributed.

Least Square Estimation(LSE)​

Least Square Estimation is a method to estimate the unknown intercept β0\beta_0β0​ and slope β1\beta_1β1​ without using Normality assumption. The idea is to find the estimates that minimizes the sum of squared errors (SSE) Q(β0,β1)=∑i=1n(Yi−β0−β1xi)2Q(\beta_{0},\beta_{1}) = \sum_{i = 1}^{n}(Y_{i}-\beta_{0}-\beta_{1}x_{i})^{2}Q(β0​,β1​)=∑i=1n​(Yi​−β0​−β1​xi​)2. Then,

  • β0^=Y‾−β1^X‾\hat{\beta_{0}} = \overline{Y} - \hat{\beta_{1}} \overline{X}β0​^​=Y−β1​^​X
  • β1^=SxySxx\hat{\beta_{1}} = \frac{S_{xy}}{S_{xx}}β1​^​=Sxx​Sxy​​
    • Sxx=∑i=1nxi(xi−X‾)=∑i=1n(xi−X‾)2S_{xx} = \sum_{i = 1}^{n}x_{i}( x_{i} - \overline{X}) = \sum_{i = 1}^{n} ( x_{i} - \overline{X})^{2}Sxx​=∑i=1n​xi​(xi​−X)=∑i=1n​(xi​−X)2
    • Syy=∑i=1nYi(Yi−Y‾)=∑i=1n(Yi−Y‾)2S_{yy} = \sum_{i = 1}^{n}Y_{i}( Y_{i} - \overline{Y}) = \sum_{i = 1}^{n} ( Y_{i} - \overline{Y})^{2}Syy​=∑i=1n​Yi​(Yi​−Y)=∑i=1n​(Yi​−Y)2
    • Sxy=∑i=1nxi(Yi−Y‾)=∑i=1n(xi−X‾)(Yi−Y‾)S_{xy} = \sum_{i = 1}^{n}x_{i}(Y_{i} - \overline{Y}) = \sum_{i = 1}^{n}(x_{i} - \overline{X})(Y_{i} - \overline{Y})Sxy​=∑i=1n​xi​(Yi​−Y)=∑i=1n​(xi​−X)(Yi​−Y)
    • β1^=∑i=1nxi(Yi−Y‾)∑i=1nxi(xi−X‾)\hat{\beta_{1}} = \frac{\sum_{i = 1}^{n}x_{i}(Y_{i} - \overline{Y})}{\sum_{i = 1}^{n}x_{i}( x_{i} - \overline{X})}β1​^​=∑i=1n​xi​(xi​−X)∑i=1n​xi​(Yi​−Y)​
