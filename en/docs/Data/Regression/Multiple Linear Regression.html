Multiple Linear Regression

Let y=(y1,y2,…,yn)T\pmb y = (y_1, y_2, \ldots, y_n)^Ty=(y1​,y2​,…,yn​)T and X=(x1T,x2T,…,xnT)TX = (\pmb x_1^T, \pmb x_2^T, \ldots, \pmb x_n^T)^TX=(x1T​,x2T​,…,xnT​)T, where xi=(xi1,xi2,…,xip)T\pmb x_i = (x_{i1}, x_{i2}, \ldots, x_{ip})^Txi​=(xi1​,xi2​,…,xip​)T for i=1,2,…,ni = 1, 2, \ldots, ni=1,2,…,n.
